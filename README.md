# Animal-Crossing-Api
an api you can use for the times bugs or fish can be caught

All of the info is for the game Animal Crossing: New Horizons.

## Fish Data Key

**fish_name:** name of the fish

**price:** the price you can sell it for in the shop

**shadow_size:** the size of the shadow in the water

**fish_num:** the number in the index

**time:** when the fish can be caught in text format

**img:** url to the image

**n_0** Etc: what month it can be caught starting from zero for northern hemisphere towns

**s_0** ... etc : what month it can be caught starting from zero for southern hemisphere towns

**h_0**... etc : what hour the fish can be caught in military time.

## Bug Data Key

**bug_name:** Name of the bug

**price:** the price you can sell it for in the shop

**img:** url to the image

**location:** where the bug can be caught

**n_0** Etc: what month it can be caught starting from zero for northern hemisphere towns

**s_0** ... etc : what month it can be caught starting from zero for southern hemisphere towns

**h_0** ... etc : what hour the fish can be caught in military time.
